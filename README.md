# Task Manager App

A Task Manager application built using React, Context API and useReducer.

## Features
- Add new task
- Edit task
- Delete task
- Mark task as Pending/Completed
- Search task by title
- Filter tasks (All / Pending / Completed)

## Tech Stack
- React
- Vite
- Context API
- useReducer
- styled-components
